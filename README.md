# 🎓 よっ友メモ (Friend Memo)

「あの人の名前、なんだっけ…？」をなくすための、シンプルで使いやすい「よっ友」管理用メモアプリ。

<img src="./icon.png" alt="よっ友メモ アイコン" width="200" height="200">

## 🌟 概要
「よっ友メモ」は、大学の同級生やイベントで知り合った人など、よっ友の情報をタグ情報と顔アイコン付きでサッと記録するためのWebアプリケーションです。
インストール不要で、ブラウザから `index.html` を開くだけですぐに使用できます。

## ✨ 特徴
- **直感的なアイコン管理**: 豊富な絵文字からイメージにぴったりのアイコンを設定可能。
- **詳細なメモ機能**: 本名だけでなく、あだ名、特徴、タグ、詳細メモを記録できます。
- **スマートな検索・フィルタ**: 自由なワード検索に加え、タグによる絞り込みに対応。
- **表示モード切り替え**: リスト表示と、一覧性の高いグリッド表示を切り替えられます。
- **モバイルファースト**: スマートフォンでの操作に最適化されたUI設計。

## 💾 データ保存について
このアプリは**ブラウザのローカルストレージ（キャッシュ）**を使用してデータを保存します。
- サーバーにデータが送信されることはないため、プライバシーが守られます。
- 同じブラウザを使用している限り、ページを閉じてもデータは保持されます。

### データ管理（バックアップ・移行）
「データ管理」メニューから以下の操作が可能です：
- **エクスポート**: データをJSON形式でクリップボードにコピー、またはファイルとして保存。
- **インポート**: 保存したJSONファイルを読み込み、データの復元やマージ（追加）が可能です。

## 🚀 使い方
1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` を任意のブラウザで開きます。
3. Safariなどで「Add to Home Screen」から、ホーム画面に追加してください。
4. 右上の「＋新規」または右下のボタンから友だちを登録してください。
